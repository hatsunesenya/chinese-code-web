# 中文电码转换器  
Chinese Telegraph Code Converter (CTC-1983)

一个用于 **中文汉字 ↔ 中文电码（中国大陆 1983 版）** 相互转换的工具，  
面向业余无线电（HAM）交流、学习与研究用途。

---

## ✨ 功能特性

- 汉字 → 中文电码（四位数字）
- 中文电码 → 汉字
- 支持未收录字符的友好提示
- 保留数字、空格及常见符号
- 响应式布局，支持桌面与移动端
- 支持 PWA（可安装、可离线使用）
- 无需后端，纯前端实现

---

## 🖥️ 在线使用

项目已部署为 Web 应用，可直接访问

支持浏览器直接使用，安卓与桌面端可安装为应用。

---

## 📱 PWA 使用说明

在支持的浏览器中：

- **桌面端（Chrome / Edge）**  
  地址栏点击“安装”按钮即可作为独立应用使用

- **安卓手机**  
  菜单 → “添加到主屏幕”

- **iOS（Safari）**  
  分享 → “添加到主屏幕”

安装后可在 **无网络环境下使用**。
