中文电码转换器

关于本软件

软件名称：
中文电码转换器（Chinese Telegraph Code Converter）
功能简介：
本工具用于实现中文汉字与中文电码（参考中国大陆 1983 版电码表）的相互转换， 适用于业余无线电（HAM）交流、学习与研究用途。
